Windows Forensics Home Lab

2024

I built a Windows forensics lab to practice memory and disk analysis on virtual machines, simulating incident response scenarios. I captured memory dumps and disk images from Windows environments and used tools such as Volatility 3, RegRipper, and Timeline Explorer to extract forensic artifacts. This allowed me to reconstruct user activity, identify persistence mechanisms, and analyze evidence of malicious behavior. The lab gave me a deeper understanding of digital forensics methodology, chain of evidence considerations, and the practical workflow of analyzing compromised Windows systems.
